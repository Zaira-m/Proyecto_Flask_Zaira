# Proyecto Flask - Programación Web

Este Proyecto fue desarrollado como parte del **Encargo 3: Fundamentos de Python con Flask**

## Descripción 
La aplicación está creada con el Framework **Flask** e incluye un menú principal y dos ejercicios:
1. **Ejercicio 1:** Calcula el promedio de notas y determina si el estudiante aprueba o reprueba según su asistencia.
2. **Ejercicio 2:** Identifica cuál de tres nombres ingresados es el más largo.

## Estructucra del proyecto
- `main.py` - Archivo principal que ejecuta la aplicación Flask.
- `templates/` - Contiene las páginas HTML del proyecto.
- `static/` - Contiene los archivos CSS para el diseño visual.
- `requirements.txt` - Dependencias necesarias para ejecutar el proyecto.

## Tecnologías utilizadas 
- Python 3
- Flask
- HTML, CSS
